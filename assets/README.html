1 Locate the File

First time only:
  • Right-click on Mac_Users_Click.command

After the first time, simply double-click the file!

Troubleshooting

❌ Browser doesn't open automatically
Manually open your browser and go to:
http://localhost:5550
❌ Port 5550 is in use
The server will automatically find another available port.
Check the terminal window for the actual URL.
❌ Questions don't load
Verify these folders exist:
  • Data/ folder with question banks
  • js/ folder with required files
  • index.html in the root
